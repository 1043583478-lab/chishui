OPERATOR PIPELINE

八算符链

不可跳步的推理编译管道。前门不稳，后段飘。从当前最早缺失的算符开始。

1. investigate→ 2. contradict→ 3. stage_judge→ 4. concentrate→ 5. align→ 6. communicate→ 7. execute→ 8. review ↻ loop

Kernel（稳定核 · 跨文明收敛 · 变化极慢）

🔍

investigate

调查研究与对象校准 · 位置 1/8 · 前置: 无

结论产生于调查的末尾。区分结构事实与局部表象，建立最小证据入口。

⚡

contradict

主要矛盾与结构诊断 · 位置 2/8 · 前置: investigate

复杂局面的解剖刀。区分纸面关系和现实控制点，找转化条件。

🗺

stage_judge

阶段判断与趋势推演 · 位置 3/8 · 前置: contradict

防御→相持→反攻。速胜论和亡国论犯的是同一个错误——片面性。

💪

concentrate

力量对比与资源集中 · 位置 4/8 · 前置: stage_judge

支点优先级 > 底盘 > 亮点。战略持久 + 战役速决，两层同时成立。

Application（应用层 · 场景组合 · 随场景演化）

🤝

align

联盟边界与关系分层 · 位置 5/8 · 前置: contradict + stage_judge

把朋友搞多，把敌人搞少。不要求所有问题上一致。

⚔

communicate

沟通策略与表达治理 · 位置 6/8 · 前置: align

有理（自卫）· 有利（胜利）· 有节（休战）。到什么山上唱什么歌。

🎹

execute

执行路线与组织换挡 · 位置 7/8 · 前置: concentrate + communicate

中心工作只有一个。十指都动，不能同时按。每步带观察信号。

🔄

review

实践检验与纠偏复盘 · 位置 8/8 · 前置: execute · 回流入口

去粗取精、去伪存真、由此及彼、由表及里。第一次几乎一定不对。

#算符类型签名前置核心价值
1investigateFuzzySituation → CalibratedObjects无结论产生于调查的末尾
2contradictCalibratedObjects → ContradictionMap← 1找主要矛盾 + 主要方面
3stage_judgeContradictionMap → StageVector← 2防御/相持/反攻 + 转段信号
4concentrateStageVector → ForceDistribution← 3选唯一方向，造局部绝对优势
5alignForceDistribution → AllianceMap← 2,3分清盟友/中间力量/对手
6communicateAllianceMap → CommunicationStrategy← 5有理有利有节 + 团结批评团结
7executeCommunicationStrategy → ActionChain← 4,6弹钢琴 · 中心工作只有一个
8reviewActionResult → CorrectionVector← 7实践检验 → 回流到 1/2/3
