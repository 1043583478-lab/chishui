## 先假设一个场景

凌晨两点。在你反复的煎熬之中，你打开了某 AI的对话框。

> **你：** 我好累。团队方向不太对，资源也快散了。帮帮我。
>
> **AI：** 面对复杂局面，建议你聚焦核心业务、梳理优先级、加强团队沟通。每一次挑战都是成长的机会 💪

随即你失望地把窗口关了。

**不是它说错了。是你俩根本不在一个频道上。** 它不知道你团队到底几个人。不知道钱还能烧多久。不知道上周谁在会议室拍了桌子。它只是从你话里抓了"团队""资源""方向"三个词，然后从概率里捞了一段最安全的东西丢给你。

也许是你忘记了，也许是你太依赖他给你解脱的思考，但我的朋友，这是算命。不是思考。

---

现在换到赤水的频道。

> **你：** 我好累。团队方向不太对，资源也快散了。帮帮我。
>
> **赤水：** 同志，先莫忙下结论。你说的"方向不对"——最近一次让你心里咯噔一下的具体事情是什么？
> A. 产品推了几个月没人用
> B. 投资人明确说赛道不行
> C. 核心成员开始流失
> D. 你自己越来越提不起劲
> 或者你补一句：
>
> **你：** C。上个月走了两个骨干，剩下的也不太稳。
>
> **赤水：** 好，情况清楚一些了。那第二件——你现在手上能动的，最吃紧的是钱、是人、是时间、还是关系？
>
> **你：** 钱还有大半年。人的问题更大——我找不到能顶上来的人。
>
> **赤水：** 同志，我先帮你把眼下这摊事掰开。你以为你的问题是"资源不够"，但我看，你真正卡住的是——团队里名义上你是拍板的人，实际上每个人在等你自己想清楚，而你自己也还没想清楚。这就是根子。
>
> 你现在的阶段不是快完蛋了，是相持。最难熬、最磨人、最多人在这步自己先泄了气。
>
> 眼下先别这也想要那也想抓。选一个方向——先把团队里剩下的人拧成一股绳。别的短板，暂时认亏。
>
> 第一步：这周先找留下来的每个人单独聊一次。不是画饼——是问他们"你觉得现在最大的问题是什么"。聊完先别给方案，把话带回来，我们再往下拆。

**看到差别了吗？**

一个急着给你答案。一个先搞清楚你到底是什么情况。

一个为了对而说正确的废话。一个不为了对，而是逼你做出正确的选择。

## 凭什么赤水能做到？

因为它的思考流程，是用代码和逻辑架构的的。8 步管道。一步咬一步：

```
 🔍                  ⚡                    🗺️
 investigate  ──→  contradict  ──→  stage_judge
 先把事实搞实       揪出根子上的主要矛盾   判断你眼下到哪一步

 💪                  🤝                    ⚔️
 concentrate ──→     align        ──→  communicate
 选一个方向打穿      分清敌友拉拢中间派     把话说到人心里

 🎹                  🔄
 execute   ──→       review
 拆成一件件动作       打完回来对账
                     对了接着走，歪了调头
```

**每条规则都是硬的。** 没搞清事实不许分析。没判明阶段不许给方案。提问有预算——问完不能再问，必须出判断。

每一步是一个 typed Python 函数。docstring 就是 LLM 执行说明书。

换 Anthropic。换 DeepSeek。换 OpenAI。换个 `llm_fn` 就完了。推理逻辑纹丝不动。

> **Kernel 永远不变。模型只是运行时。**

| 你的卡点 | 他的对策 |
|---------|------|
| 🌀 脑子一团浆糊，不知道从哪想 | 先把"事实"和"脑补"掰开 |
| 💥 感觉哪哪都是问题 | 揪主要矛盾——"解决它，别的连带松动" |
| 🐘 对手像座山，觉得自己没戏 | 把所有牌摊桌上。看趋势，看质，不看量 |
| 🍕 资源一点点，什么都放不下 | "十指都动，同时按不成调子"——逼你选一个方向 |
| 🏁 事做完了，心里没底 | 拿判断跟结果对账。"第一次几乎一定不对，修正后才对" |

不给万能答案。给你一套**从泥里把答案刨出来的办法。**
